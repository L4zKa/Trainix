# Trainix

Trainix ist eine Progressive Web App (PWA) zum Tracken von Krafttraining-Workouts.

Du kannst eigene Workout-Templates für einzelne Trainingstage erstellen und diese im Gym direkt starten.  
Alle absolvierten Einheiten werden automatisch gespeichert und können später in der History (noch in entwicklung) eingesehen werden.

---

## Funktion

- Eigene Workout Templates erstellen
- Training direkt aus Templates starten
- Sets & Wiederholungen tracken
- Workout History
- Installierbar als PWA (Desktop & Mobile)
- Offline Support (basic)

---

## Installation

Trainix läuft direkt im Browser:

https://l4zka.github.io/Trainix/

### Als App installieren

1. Link in Chrome oder Edge öffnen
2. „Installieren“ oder „Zum Startbildschirm hinzufügen“ wählen
3. Trainix startet danach wie eine native App
